Codify( )

Code in real time using Python or JavaScript and work online with anyone, anywhere, anytime!

Description

Description


Codify is an online editor that lets you code in real time using Python or JavaScript as programming languages.
It lets you work with anyone, no matter where you are, and no matter what time it is.

The purpose of Codify is to bring people together in a quite understandable editor, and thus create a comfortable workspace where everyone feels happy and at ease with their work.

"Codify not only wants to change the way people code, but the way code is shared around the world."
